# プロジェクト概要

- Expo + Expo Routerで構築されたReact Nativeアプリ（TypeScript）。
- 主要ディレクトリ:
  `app/`（画面とルーティング、タブ構成）、`components/`（再利用コンポーネント）、`constants/`、`context/`、`hooks/`、`data/`、`assets/`、`scripts/`（リセットスクリプト）。
- エントリポイント:
  `app/_layout.tsx`および`app/(tabs)/_layout.tsx`からルート構成。
- 目的詳細は未記載（READMEはExpoテンプレート）。画面/機能を読む際は`app/`配下を確認。
